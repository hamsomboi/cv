HYPER TEXT MARKUP LANGUAGE (HTML)



HTML is used for structuring content (text, image, hyperlinks) through use of markup language

Useful HTML Tags

HTML Tags Tag Description Tag attributes
Headers Tag (h1 - h6) Define heirachy of texts
Paragraph Tag (p) Define a new paragraph
List Tag (ul, ol, li) Define ordered (ol) or unordered (ul) lists and items in the list type="i/a/1"
start="5"
Line Break (br) Self-closing tag that inserts a line break
Horizontal Rule (hr) Self-closing tag that inserts a horizontal line size="3"
noshade
Style changing Tag (em, strong, b, i) Define tags that require emphasis/strength or change style to bold/italics
Image Tag (img) Self-closing tag that inserts image src="image source directory"
alt="Text to be displayed if source broken"
Anchor Tag (a) Define hyperlinks that can direct user to another site href="link directory"
Table Tag (table, thead, tbody, tr, td) Define a table
Form Tag (form, label, input, textarea) Creates a form for user to interact with action="mailto:add@" - Defines submit button clicked
method="post"
enctype="plain/text", type="text/color/submit"
Style Tag (Internal CSS) Allows defining of selector to change their style properties using CSS language selector{property: value}

Class can be assigned to a tag element to group content with similar properties

ID can be assigned to a tag element to define unique content

The heirachy for defining the style properties of an element is (In-line > Internal CSS > External CSS) and (ID > Class > Tag)
